# Blue Guardian – GitHub Pages Site

This repository contains a small static website for hosting the Privacy Policy, Terms & Conditions, and Data Safety summaries for the Blue Guardian app (Bournemouth University).

## Structure
- `index.html` – Privacy Policy
- `terms.html` – Terms & Conditions
- `data-safety.html` – Google Play Data Safety summary
- `appstore-summary.html` – App Store privacy summary
- `contact.html` – Contact page
- `assets/style.css` – Stylesheet

## How to deploy
1. Create a new public GitHub repository (for example `blueguardian-privacy`).
2. Upload the files and the `assets/` folder to the repository root.
3. In the repository settings enable **Pages** and select `main` branch / root folder.
4. The site will be available at `https://<your-github-username>.github.io/<repo-name>/`.
